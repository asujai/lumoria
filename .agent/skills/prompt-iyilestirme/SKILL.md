# Prompt Optimization Skill

When improving a prompt, follow these steps:

## Optimization checklist

1. **Clarity**:
   - Is the goal clearly defined?
   - Are instructions ambiguous?

2. **Specificity**:
   - Are output expectations defined?
   - Is format specified?

3. **Structure**:
   - Is the prompt logically organized?
   - Are sections separated clearly?

4. **Constraints**:
   - Are limits defined (length, tone, format)?
   - Are assumptions removed?

5. **Context Enrichment**:
   - Is enough background information provided?
   - Is the role of the AI clearly assigned?

## How to provide output

- Show original prompt
- Show improved prompt
- Explain improvements briefly
- Keep improvements minimal but impactful
